z-index

To place an element behind another, we set "z-index" to a lower number. In this case is "-1"

(to be added on CSS) .z-1{ width:100px; position: absolute; z-index: 0;}

The higher number z-index:1 or z-index:2 will always be in front. 1, 2, etc, to make it overlap.
